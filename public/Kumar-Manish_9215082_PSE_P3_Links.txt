APPLICATION OVERVIEW
--------------------
ConciergeX is a premium luxury services marketplace web application built with Next.js 16, React 19, TypeScript 5, Prisma ORM, and Tailwind CSS 4. It features:
- 90 luxury services across 16 categories
- Three user roles: Admin, Client, Service Provider
- Service booking with mock payment processing (15% platform fee)
- Review and rating system
- Real-time messaging
- Analytics dashboards with charts
- Responsive dark luxury theme with gold accents
- 15 RESTful API endpoints
- 9 database models with proper relationships

TECHNOLOGY STACK
----------------
- Frontend: React 19, Next.js 16, Tailwind CSS 4, shadcn/ui, Framer Motion, Zustand
- Backend: Next.js API Routes, Prisma ORM, bcryptjs
- Database: SQLite (development), PostgreSQL-ready for production
- Tools: Bun, ESLint, GitHub, Docker, Caddy
